# BigTech Hotel Seller (demo)

Proyecto muy simple con Node.js + Fastify.

QUE HACE
- API: GET /api/products (devuelve los productos de data/products.json)
- Web: sirve archivos desde la carpeta public (por defecto public/index.html)

REQUISITOS
- Tener Node.js instalado

COMO EJECUTAR
1) Instalar dependencias:
npm install

2) Arrancar servidor:
node server.js

3) Abrir en el navegador:
- Web: http://localhost:3000
- API: http://localhost:3000/api/products

ESTRUCTURA (TIPICA)
- server.js
- data/products.json
- public/index.html
- public/app.js
- public/styles.css
